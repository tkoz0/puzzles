# lists commands for a python scrypt to run to fix errors in the data
# lines that start with '#' are ignored (comments)
# comments must be their own line

# the scrypt will check if the .old file exists and verify md5sums if so
# if the .old file doesnt exist, it will perform modifications and check md5sums
# md5sums are a check against the website changing and invalidating edits
# file <path> should not include 'puzzle_data/' prefix

# commands are in the form of file fixes structured as below
# file <path>
# md5sums <old> <new>
# <edit commands...>
# end

# the following edit commands are supported:
# add_begin_line
# add_end_line
# edit_line <index> <new_contents>
# insert_line <index> <contents>
# delete_line <index1[,index2,...]>

# for insert_line, index can be negative for python negative indexing
# for delete_line, indexes can be a single number or a range 'm-n' with m < n
# for index ranges, negative numbers are not supported

# -- Heyawake --

# 'send' instead of 'end'
file Heyawake/576.a.txt
md5sums c4712ff36379afcce7a290aa004a1cdd 429e37169726f6261eadec81c8c48b07
edit_line -1 end
end
